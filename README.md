# 🤖 Global AI Impact Dashboard - Data Viz & Storytelling

Este projeto foi desenvolvido para explorar o impacto da Inteligência Artificial em diferentes setores e países, com foco total em **Data Visualization** e **Experiência do Usuário (UX)**. 

O objetivo principal foi transformar métricas de adoção e confiança em uma narrativa visual intuitiva, utilizando o **Figma** como base de design.

## 🛠️ Implementação Técnica (Power BI)
Apesar do foco em design, a estrutura de dados foi pensada para ser organizada e escalável:

* **Organização do DAX:** Utilização de *Display Folders* (Pastas) para separar métricas de negócio (`AI Metrics`) de indicadores sociais (`Human Indicators`).
* **Padronização:** Nomenclatura em *snake_case* e prefixos (`avg_`, `sum_`, `div_`) para facilitar a manutenção técnica.
* **Interatividade:** Navegação fluida via botões e filtros personalizados (Chiclet Slicer).

## 📊 Métricas Analisadas
Para manter o Storytelling claro, foquei em indicadores diretos:
* **Adoption Rate:** Penetração da IA por indústria.
* **Revenue Increase:** Impacto financeiro direto.
* **Consumer Trust:** Nível de confiança da sociedade.
* **Job Loss Risk:** Análise de risco ocupacional.
